# MicroScore Community

Application de microfinance communautaire avec scoring de crédit basé sur des données alternatives.

## Technologies

### Backend
- Java 21
- Spring Boot
- Spring Cloud
- Eureka Server
- Spring Gateway
- JWT

### Frontend
- Angular

### Base de données
- MySQL

## Architecture

Microservices :

- user-service
- loan-service
- repayment-service
- scoring-service
- admin-service
- api-gateway
- discovery-service

## Equipe

Projet réalisé dans le cadre du projet académique de fin d'année.
